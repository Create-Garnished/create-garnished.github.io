Drops from Hatchets - Nether

Hatchets allow for a multitude of drops to become accessible to players.

Enflamed Mandible can be dropped by Blazes when using the Ravaging enchantment on a hatchet.

Ghast Tendril can be dropped by Ghasts when using the Ravaging enchantment on a hatchet.

Molten Remnant can be dropped by Magma Cubes when using the Ravaging enchantment on a hatchet.
