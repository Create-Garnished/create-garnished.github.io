Bulk Dyeing

Coloured liquefied mastic resin can be used to redye a multitude of items, including:

  • Wool
  • Glass & Glass Panes
  • Banners
  • Candles
  • Shulker Boxes
  • Mastic Resin
  • ...

[2024-03-22] Future Plans

In the future, there are plans to eventually add unique recipes to each individual bulk dyeing recipe. For example, you might be able to make slime by bulk dyeing dough with the green liquefied mastic resin in a later version.

There are also plans to add mod integration/compatibility for dyeable/coloured items. For example, canvas signs from Farmer's Delight might receive the ability to be redyed via this method.
